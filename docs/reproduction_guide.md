# Reproduction Guide

This guide explains how to reproduce the analysis from scratch using the
materials in this repository.

---

## 1. Prerequisites

- Git
- Conda (or Mamba) for environment management
- Internet connection to download raw data (if not already present)
- A UNIX-like shell (Linux, macOS, or WSL on Windows recommended)

---

```bash
git clone <YOUR_REPO_URL>.git
cd <YOUR_REPO_NAME>
